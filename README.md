### Player

 Features:
  - Control: Play/pause.
  - Control: Slide to any seconds.
  - Control: Loop on/off.
  - Control: Next/Prevous song in list (dynamic). Button is disable if current song is first/last.
  - Display: Timer.
  - Display: Dynamic background folloing song image.
  - Display: Spinning disc when playing.
  - Add to playlist (Incomming feature ‚è∞)

### Main screens

 Features:
  - Display: Skeleton when data is loading.
  - Display: Glassmorphism GUI.
  - Display: Scrollable cards.
  - Content: Songs, Artists, Albums.
  - Add to playlist (Incomming feature ‚è∞).
  - Add to favorites üíö (Green heart), 
  - Keep your favorite songs in device.
  - Quick Player: listen everywhere with floating player while you are browsing.
  - Artist page & Album page: dynamic glassmorphism background following artist/album image.
  
Content:
